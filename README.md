# AniList API Project

A web application that uses the AniList API to fetch and display user data.

## Built With

* HTML
* JavaScript
* CSS

## Instructions

1. Clone the repository
2. Open `index.html` in your web browser
3. Enter a valid AniList user ID in search bar
4. Click "Search"
